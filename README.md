<br/></br>### 课程介绍:
量化投资是指通过数量化方式及计算机程序化发出买卖指令，以获取稳定收益为目的的交易方式。在海外的发展已有30多年的历史，其投资业绩稳定，市场规模和份额不断扩大、得到了越来越多投资者认可。
事实上，互联网的发展，使得新概念在世界范围的传播速度非常快，作为一个概念，量化投资并不算新，国内投资者早有耳闻。但是，真正的量化基金在国内还比较罕见。同时，机器学习的发展也对量化投资起了促进作用。
